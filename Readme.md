## Bayesian Quantile Regression with Variable Selection

Here we provide the script BQRVS.R. The script implements Bayesian
Quantile Regression with Variable Selection. 

In order to run the script you need:

- R (https://www.r-project.org)
- R packages GIGrvg and BGLR.

The script includes a cople of examples that shows how to use the software.
